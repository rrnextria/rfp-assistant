# Plan & Report Templates

## Templates

| Template | Use Case |
|----------|----------|
| `simple_plan_template.md` | Single-file plan for small scope (1-5 tasks) |
| `master_plan_template.md` | Central index for complex multi-phase projects |
| `phase_plan_template.md` | Per-phase document within a complex plan |
| `code_complete_template.md` | Code implementation artifact for coder-reviewer loop |
| `report_templates.md` | Implementation and review report formats |

## Choosing Simple vs Complex

- **Simple:** One subsystem, low coordination, 1-5 tasks
- **Complex:** Multi-phase, cross-module, many tasks
